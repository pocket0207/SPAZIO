Shodan 검색 도구는 사용자 입력을 최소화하여 검색 쿼리를 특수하게 만들어 사용자를 지원합니다.

사용자는 다음을 수행할 수 있습니다.:

  • 네트워크에서 공개적으로 사용 가능한 모든 장치를 검색k
  • 각 호스트에 특정한 취약점 식별
  • 포트, 소프트웨어 유형, HTTP 상태 코드, 취약성 또는 키워드별로 네트워크에서 장치 검색


Shodan 이란?

간단하게 설명하면 Shodan은 사물 인터넷 (IoT)을위한 검색 엔진입니다.

웹 페이지에서 데이터를 크롤링하고 웹 컨텐츠 색인을 작성하는 Google과 달리 Shodan은 주로 포트를 찾은 다음 결과 배너를 가져와 색인합니다. 특히 Shodan은 전 세계의 분산 스캐너를 사용하여 대상 IP 주소를 임의로 선택하고 수신 TCP 및 UDP 포트를 식별합니다. 프로토콜 배너, 웹 페이지 및 기타 서비스 데이터를 수집하기 위해 수신 포트가 추가로 열거됩니다. 이 모든 데이터는 인터넷 장치의 "무엇"을 설명하는 검색 가능한 거대한 데이터베이스에 추가됩니다.
